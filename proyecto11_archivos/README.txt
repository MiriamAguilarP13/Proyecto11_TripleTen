Proyecto 11 - Sprint 11: Automatización

Descripción del Proyecto:
Se proporciona un Dataset llamadado trending_by_time.csv, que tiene información de vídeos publicitarios en la agencia 
de publicidad Sterling & Draper. Se dedica mucho tiempo a analizar tendencias de vídeos en YouTube para determinar qué 
contenido merece atención para la mercadotecnia.

Objetivo: 
Realizar un Dashboard en Tableau con los contenidos, diseño y los datos que se deben presentar, de acuerdo a lo acoradado
con el equipo de marketing.


Pasos para ejecutar los archivos.
1. Hacer doble clic en el archivo con el nombre link.py, esto abrira a la página web del Dashboard en el servidor de Tableau Public.
	* Si la acción anterior no abre la página web, puedes abrir una terminal en la carpeta donde se encuentran los archivos y 
	  escribir `python link.py` seguido de "Enter". Esta operación iniciará la apertura de la página web.

2. Abrir el archivo PDF 3_Proyecto_11_tendencias_videos, es una presentación con los resultados más relevantes del Dashboard
y que resultan de interés para el equipo de marketing.
